# react-native-masonry-flatlist

Pure js Masonry Flatlist

## Installation

```sh
npm install react-native-masonry-flatlist
```

## Usage

```js
import { multiply } from "react-native-masonry-flatlist";

// ...

const result = await multiply(3, 7);
```
